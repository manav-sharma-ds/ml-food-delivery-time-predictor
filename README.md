# 🍔 ML Food Delivery Time Predictor

A Machine Learning web application built using **Python** and **Streamlit** that predicts food delivery time based on delivery distance, customer rating, order type, and vehicle type.

## 🚀 Live Demo

https://ml-food-delivery-time-predictor-2g8cetp6ssgmxwxrg4sqp2.streamlit.app/

---

## 📌 Features

- Predicts estimated delivery time
- Interactive Streamlit interface
- Random Forest Regression model
- Supports multiple order types
- Supports multiple vehicle types
- Fast and accurate predictions

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

---

## 📊 Machine Learning Model

- Algorithm: Random Forest Regressor
- Trees: 400
- Max Depth: 25
- Train-Test Split: 80% / 20%
